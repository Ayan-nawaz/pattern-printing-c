# Pattern Printing in C

This repository contains basic pattern printing programs written in the C programming language. These programs are useful for beginners to improve their understanding of loops, nested loops, and pattern logic.

## Programs Included

- Alphabet Square Pattern
- Star Cross Pattern
- Floyd's Triangle
- 0 & 1 Triangle
- Star Plus Pattern

## Concepts Covered

- Nested Loops
- Pattern Printing
- Character Patterns
- Number Patterns
- Star Patterns

## Language

- C

## How to Run

1. Clone this repository.
2. Open the project in Visual Studio Code or any C compiler.
3. Compile the desired program.
4. Run the executable.
